# @d1y/kitty

仅用于展示小猫源编写，方便参考

![Image 350x314.gif](https://s2.loli.net/2025/10/12/EVSOIbTzfWkoNB7.gif)

`kitty` / `@types/kitty` 来自上游仓库，需先 clone 再安装：

```bash
git clone --depth 1 --branch dev \
  https://github.com/waifu-project/movie.git vendor/movie
bun install
bunx kitty-parse -o result.json
```
